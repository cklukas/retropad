retropad

A lightweight Notepad-style text editor for Windows. Supports Unicode text files with BOM detection, word wrap, status bar, find/replace, and printing with customizable headers, footers, and margins.

Basics

  • New/Open/Save: Standard file operations. Encodings honored via BOM.
  • Word Wrap: Toggle to wrap lines and hide horizontal scrolling.
  • Status Bar: Shows line, column, and total line count.
  • Find/Replace: Ctrl+F / Ctrl+H with Match Case and direction.
  • Go To: Jump to a specific line when word wrap is off.
  • Font: Choose the editor font via Format > Font.
  • Time/Date: Inserts the current time and date.

Printing & Page Setup

Use File > Page Setup to edit margins and header/footer text. Default margins are 0.75" left/right and 1.0" top/bottom. Default header is the file name; footer is “Page X of Y”.

Header/Footer Formatting Codes

  • &l start left-aligned section
  • &c start centered section
  • &r start right-aligned section
  • &d current date
  • &t current time
  • &f file name
  • &p current page number
  • &P total pages
  • && literal ampersand

Example: &l&f&cPage &p of &P&r&d places the file name left, page count centered, and date right.
